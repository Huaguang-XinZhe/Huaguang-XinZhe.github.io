    当前的核心是 求职！只要有助于在截至日期前达成目标，干什么，怎么干都行！

    所以，可以自由一些，别固化、限制自己！

    那我求职的核心是什么？

    • 立马能干活，做后端相关的开发，懂前端，愉快对接；
    • 有自己的独立项目（网站 + Android）和个人博客，能流畅访问；
    • GitHub 活跃，参与、独立开源；
    • 认真、善总结、常输出，有不断更新的知识库和稳定输出的自媒体账号；
    • 同价值（关注科技，追求效率），低要求（薪资、待遇等），极稳定（只要能收获成长，3 年内不离职），高性价比；

    大方向是，这个月先搞定后端（也就是那套 Java Web 课程），下个月开始项目（主要是了解思路、流程、工具等），同时结束前端（JS 和 Vue）。

    不过也不绝对，还是容许变化的。

    只要不断地向 “最终目标” 推进就成！

    –

    “总结、输出 -> 构建知识库 -> 自媒体稳定输出” 流程现在就可以开始了。

    它也是目标的一部分。

    –

    GitHub 可以好好清整一下，准备开源了，就从 Markdown 主题（Typora、Obsidian、微信公众号）开始吧。

    –

    个人博客可以跟进升级一下，部署到自己的服务器，用上 CDN 了。

    Linux 相关知识也可以跟着学起来了。

    –

    仪表盘（网页），明天时间统析后就可以开始做了，就算是现实应用。

    部署到个人服务器，财务也可以结合进来。

    –

    伴学实践 是我所缺乏的，这方面需要重点关注、落实。

    不能全是理论和实践，也需要有 现实应用，否则，我会感觉无处着力，产生一种虚无感。

    每天保证 3 个小时的伴学实践，其他的不做限制，只要是向着 “最终目标” 推进就成。

    为了了解进度、统筹安排，每天就有必要进行详尽的时间统析，并将数据汇总到 仪表盘 当中。

    这个仪表盘目前是不存在的，我需要去创建它！

    想做，为什么不让自己做？趁着惯性，顺势而为，既愉快又轻松，一点也不需要自律。

    只要是不断地向目标靠近，最终（在截止日期之前）实现目标就成了嘛！

    –

    自由开始，事后定调。

    不要自己束缚自己！
